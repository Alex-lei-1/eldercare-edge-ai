# ElderCare Edge AI — project site

On-device **fall detection** + a fully local **voice companion** on the RK3588 (Orange Pi 5 Plus).

Real-time pose-based fall detection on a 6-TOPS NPU, plus an offline
wake-word → ASR → LLM → TTS voice assistant. **All inference runs on the edge — no cloud.**

**Live site:** https://alex-lei-1.github.io/eldercare-edge-ai/

---

This repository hosts only the public project showcase (a single static `index.html`).
The implementation lives in a separate private repository.
